
# Libvirt-mang

This is a library for managing libvirt using the rust bindings.

Currently the roadmap that is going to be supported is:
- starting vm's in libvirt
- checking if the platform supports libvirt
- destroying vm's
- and more quality of life commands, such as networking and ssh

if you would like more info go to the wiki

[qemu-homelab-wiki](https://github.com/lucassimpson0213/sys-qemu-homelab/wiki)
